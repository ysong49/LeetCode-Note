## 254. Factor Combinations [M]
Numbers can be regarded as product of its factors. For example,
```
8 = 2 x 2 x 2;
  = 2 x 4.
```
Write a function that takes an integer n and return all possible combinations of its factors.

#### Note: 
  1. You may assume that n is always positive.
  2. Factors should be greater than 1 and less than n.

#### Examples: 
input: `1`
output: 
```
[]
```
input: `37`
output: 
```
[]
```
input: `12`
output:
```
[
  [2, 6],
  [2, 2, 3],
  [3, 4]
]
```
input: `32`
output:
```
[
  [2, 16],
  [2, 2, 8],
  [2, 2, 2, 4],
  [2, 2, 2, 2, 2],
  [2, 4, 4],
  [4, 8]
]
```

## Code:
```c++
class Solution 
{
    // date: 2016-08-30     location: Santa Clara City Library
public:
    vector<vector<int>> getFactors(int n) 
    {
        vector<vector<int>> res;
        helper(res, {}, n);
        res.pop_back();
        return res;
    }
    
    void helper(vector<vector<int>>& res, vector<int> factors, int num)
    {
        if (num == 1)
        {
            res.push_back(factors);
            return;
        }
        
        for (int i = 2; i <= num; i ++)
        {
            if (num % i == 0 && (factors.empty() || factors.back() <= i))
            {
                factors.push_back(i);
                helper(res, factors, num / i);
                factors.pop_back();
            }
        }
    }
};
```
