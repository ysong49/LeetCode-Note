# 65. Valid Number [H]
Validate if a given string is numeric.

Some examples:
```c++
"0" => true
" 0.1 " => true
"abc" => false
"1 a" => false
"2e10" => true
```

# Note: 
It is intended for the problem statement to be ambiguous. You should gather all requirements up front before implementing one. 


# Code:
```c++
class Solution {
public:
    bool isNumber(string s) 
    {
       int i = 0;
    
        for (; s[i] == ' '; i++);
    
        if (s[i] == '+' || s[i] == '-') 
            i++; 
    
        int n_nm, n_pt;
        for (n_nm=0, n_pt=0; isdigit(s[i]) || s[i]=='.'; i++)
            s[i] == '.' ? n_pt++ : n_nm++;       
        if (n_pt > 1 || n_nm < 1) 
            return false;
    
        if (s[i] == 'e')
        {
            i ++;
            n_nm = 0;
            if (s[i] == '+' || s[i] == '-')
                i ++;
            while (isdigit(s[i]))
            {   
                i ++;
                n_nm ++;
            }
            if (n_nm < 1)
                return false;
            
        }
    
        for(; s[i] == ' '; i++);
    
        return i == s.size();
    }
};
```
